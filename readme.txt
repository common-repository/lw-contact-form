Addon for Contact Form 7. The plugin saves contact form 7 submissions to your WordPress database.
By simply installing the plugin, it will automatically begin to capture form submissions from contact form 7 and automaticly show them on Localweb's app "My Localweb".

*Note: The plugin uses services from https://localweb.it/ and is intended ONLY for <a href="https://localweb.it/" alt="Localweb">Localweb srl</a> clients.

== Description ==

This plugin creates a new table on the database named prefix->inserimeti_cf witch saves all the Contact Form 7 submits.
The plugin then sends this data to https://localwebapi.ids.al/contactFormWeb (one of Localweb's API servers)  the LocalWeb's app <a href="https://itunes.apple.com/al/app/my-localweb/id1094286698" alt="App Store">App Store</a> and <a href="https://play.google.com/store/apps/details?id=it.localweb" alt="Play Store">Play Store</a>.

*Note: The plugin uses services from https://localweb.it/ and is intended ONLY for <a href="https://localweb.it/" alt="Localweb">Localweb srl</a> clients.


== Installation ==


1. Upload the plugin folder lw-contact-form-7-addon and all it's contents to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. There is no need to take any other stes because the plugin will work by itself on the backgrownd.
